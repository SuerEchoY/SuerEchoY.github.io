个人主页

本页面主要归档我本科及研究生期间的核心学术资料与工程项目，涵盖从底层硬件驱动到上层机器人算法的全栈开发。

🤖 机器人系统与算法 (Robotics)

1. 全栈导航系统框架 (ROS/ROS2)

独立架构并开发的机器人导航系统，覆盖从感知到运动控制的完整闭环。

  • 建图与定位： 融合激光雷达与里程计数据的 SLAM 实现。
  • 路径规划： 实现了全局规划器（Global Planner）与局部避障算法（Local Planner）。
  • 技术栈： C++, ROS/ROS2, CMake

2. Unitree Go2 四足机器人二次开发

基于 Unitree SDK 进行上层运动控制开发与算法验证。

  • 轨迹追踪： 解析给定轨迹数据，通过 SDK 下发底层控制指令，实现高精度轨迹跟随。
  • Sim2Real 验证： 结合 NVIDIA Isaac Lab 进行强化学习算法的仿真与真机部署测试。

⚙️ 嵌入式与底层驱动 (Embedded & Linux)

1. 嵌入式 Linux 驱动开发

  • 基于 ARM 架构的底层驱动编写与内核裁剪。
  • 熟悉设备树配置及 I2C/SPI 等常用总线协议，确保传感器数据的高效采集与上下位机通信。
  • 技术栈： STM32, Linux Kernel, C

🎛️ 硬件加速 (FPGA)

1. 高速接口与图像处理流水线

利用 FPGA 的并行计算优势，实现实时机器视觉预处理。

  • 图像流处理： 实现了基于 Verilog 的图像滤波、边缘检测及色彩空间转换流水线。
  • 高速传输： 负责高速接口的时序收敛与数据缓存设计。
